# Чек-лист / Тест-кейсы: Основной бизнес-процесс ЭТрН

## Легенда приоритетов
- **P0 (Critical):** Блокирует бизнес-процесс, невозможно выполнить перевозку
- **P1 (High):** Критично для корректного функционирования, но есть обходной путь
- **P2 (Medium):** Важно для UX/корректности данных, не блокирует процесс
- **P3 (Low):** Косметические/опциональные улучшения
