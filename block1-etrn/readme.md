# Блок 1: Тестирование требований (Бизнес-процесс ЭТрН)

## Исходные данные

### Контекст
Электронная транспортная накладная (ЭТрН) — цифровой аналог бумажной транспортной накладной, фиксирующей факт перевозки грузов. С 1 сентября 2026 года становится обязательной согласно №140-ФЗ от 07.06.2025.

**Обязательные титулы ЭТрН:**

| Титул | Участник | Стадия | Тип подписи | Содержание |
|-------|----------|--------|-------------|------------|
| Т1 | Грузоотправитель | Погрузка | КЭП | Стороны, условия договора, водитель, ТС, груз, сопроводительные документы |
| Т2 | Перевозчик (водитель) | Погрузка | ПЭП | Состояние груза при приёме, замечания по погрузке |
| Т23 | Перевозчик | Погрузка | КЭП | Подтверждение приёма груза перевозчиком |
| Т3 | Грузополучатель | Выгрузка | КЭП | Приём товара, состояние груза, частичная/полная приёмка |
| Т4/Т42 | Перевозчик (водитель) | Выгрузка | ПЭП | Подтверждение передачи груза, замечания |
| Т43 | Перевозчик | Выгрузка | КЭП | Закрытие рейса перевозчиком |

**Опциональные титулы:**

| Титул | Участник | Назначение |
|-------|----------|------------|
| Т5 | Перевозчик | Фиксация финальной стоимости перевозки |
| Т6 | Грузоотправитель | Подтверждение стоимости |
| Т7 | Грузоотправитель | Указание о переадресации/возврате груза |
| Т8 | Перевозчик | Изменение грузополучателя или адреса |

### Основные участники процесса
1. **Грузоотправитель** — формирует Т1, подписывает КЭП
2. **Перевозчик (ТК)** — подписывает Т2 (через водителя ПЭП), Т23 (КЭП), Т4 (через водителя ПЭП), Т43 (КЭП)
3. **Водитель** — подписывает Т2, Т42 простой электронной подписью (ПЭП)
4. **Грузополучатель** — подписывает Т3 (КЭП)
5. **Оператор ИС ЭПД (ЭДО)** — маршрутизирует документы между участниками
6. **ГИС ЭПД (Минтранс)** — верифицирует, присваивает УИД, хранит документы

### Жизненный цикл ЭТрН (основные статусы)

1. **Черновик** — документ создан, но не подписан грузоотправителем
2. **Подписан отправителем (Т1)** — грузоотправитель подписал КЭП, документ передан в ЭДО
3. **Принят к перевозке (Т2+Т23)** — водитель (ПЭП) и перевозчик (КЭП) подписали
4. **В пути** — груз перевозится; возможны переадресации (Т7, Т8)
5. **Доставлен (Т3)** — грузополучатель подписал Т3 (полная/частичная приёмка)
6. **Завершён (Т4/Т42+Т43)** — перевозчик закрыл рейс, документооборот завершён
7. **Аннулирован** — документ аннулирован по соглашению сторон

После каждого этапа данные автоматически передаются в ГИС ЭПД через оператора.
